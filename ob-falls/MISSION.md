# Mission: Master "ob" and "falls" for the B1 Exam

## Why
The user is preparing for a B1 German exam and needs to clearly distinguish between "ob" and "falls" (and likely "wenn") to improve their grammatical accuracy and speaking/writing skills.

## Success Criteria
- [ ] Correctly identify when to use "ob" (indirect questions/whether).
- [ ] Correctly identify when to use "falls" (conditions/in case).
- [ ] Understand the overlap and differences with "wenn".
- [ ] Pass B1-level practice exercises with 100% accuracy.
