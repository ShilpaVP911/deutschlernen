Lesson 1: Mastering "Ob"

In English, we use "if" for almost everything. In German, "if" is split into three main words. Today, we focus on ob.

Goal: Learn to use "ob" for indirect questions.

The "Whether" Test

The easiest way to know if you should use ob is to try and replace "if" with "whether" in your English sentence. If it still makes sense, use ob.

  • "I don't know if he is coming." → "I don't know whether he is coming." (Makes sense? Yes! Use ob.)
  • "I'll help you if I have time." → "I'll help you whether I have time." (Makes sense? No, it changes the meaning. Use wenn.)

Word Order Reminder

Ob is a subordinating conjunction. This means the verb goes to the end of the clause.

Ich frage mich, ob er heute kommt.

Check your understanding:

Which sentence uses "ob" correctly?

Ich gehe spazieren, ob die Sonne scheint.
Können Sie mir sagen, ob der Bus schon weg ist?
Ob ich müde bin, gehe ich ins Bett.
